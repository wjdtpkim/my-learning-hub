## 무엇을

4주 동안 배운 걸 쌓아둘 내 학습허브 사이트를, 코딩 없이 만들고 실제 웹 주소로 배포했어요.

## 왜

AI랑 함께 배운 걸 기록하고 공유하려면, 매번 로컬에서만 보는 게 아니라 인터넷 어디서든 접속할 수 있는 내 사이트가 필요했어요. 그리고 앞으로 4주 동안 옆에서 같이 작업할 AI 학습메이트도 이때 같이 만들었어요.

## 어떻게

1. GitHub 템플릿을 내 컴퓨터로 가져오기
2. 인터뷰 방식으로 AI 학습메이트(모닥) 만들기 — SOUL·USER·AGENTS·CLAUDE.md 4개 파일
3. Node.js 설치하고 로컬 미리보기 띄우기
4. GitHub + Vercel 가입해서 실제 웹 주소로 배포하기
